This cors proxy is very important javascript applications that need to fetch public RDF resources on the Web which are not CORS-enabled.

All the request headers passed from the client on to this one are passed on. And the resulting response headers are passed along also, but with an additional Access-Control-Allow-Origin: {value sent by remote resource, falling back to request Origin header value, falling back to *} and also Access-Control-Allow-Credentials:true Header so that the JavaScript in the client can accept the request.

Remote RDF resource: document.
