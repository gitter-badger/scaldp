Welcome to RWW-Play

You’re using Play @play.core.PlayVersion.current

About

This Web Application is putting together the tools to build the read-write-web. This requires a number of pieces:

  • The Linked Data Platform (LDP spec) defined at the W3C, to specify how a client can create, update, query and delete resources.
  • Authentication using WebID ( see WebID.info ).
  • Authorization using the Web Access Control ontology, together with WebID.

This should be complemented with the help of some Web2.0 services:

  • WebID Certificate Creation for people with existing profiles
  • A WebID Identity Provider ( such as this one ) so that other apps that don't have TLS can join easily.
  • A WebID client test service: so that authentication problems can be quickly found and understood.
  • WebID server tester: this will be useful to help people with new WebID implementations.

It will be then useful to build some simple and useful apps to demonstrate the platform such as:

  • Blog engine
  • Profile page where friends can be added
  • Pingback

The aim is to have these pieces working together to build a use case, and later seperate the pieces into autonomous libraries.

What is working now?

At this point this is still very much work in progress. But some pieces are already working.

  • WebID Certificate Creation: @routes.ClientCertificateApp.get for people that already have a WebID and wish to have an easy way to create a certificate. (does not work with IE yet ).
